# ftbadge

Generate a dynamic badge displaying a 42 profile summary from a GitHub login.

## Usage

```
[![ftbadge](https://ftbadge.cc/<login>?scale=<scale>)](https://github.com/thoguerin64/ftbadge)
```

## Exemple

[![ftbadge](https://ftbadge.cc/tguerin?scale=1.25)](https://github.com/thoguerin64/ftbadge)
